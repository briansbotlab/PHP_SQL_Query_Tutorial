# PHP_SQL_Query_Tutorial
The tutorial is about SQL database basic operations in php. 

* view.php 為使用者介面
* controller.php 負責處理使用者傳入參數
* success.php 為成功查詢之畫面
* fail.php 為查詢失敗之畫面
